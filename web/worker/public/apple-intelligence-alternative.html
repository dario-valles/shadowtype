Questions

Apple Intelligence alternative FAQ

Is Apple Intelligence free?
Yes — Apple Intelligence and its Writing Tools are free and built into macOS 15.1+ on Apple Silicon. You’d only choose Shadowtype if you want what Apple Intelligence doesn’t offer: continuous multi-sentence inline prediction in every app, a guarantee that nothing ever leaves your Mac, model choice, and macOS 14 support. If occasional rewrites in Apple’s apps are all you need, Apple Intelligence is a fine choice and costs nothing.
Isn’t inline predictive text the same as Apple Intelligence?
No — they’re separate features. macOS inline predictive text (the short ghost suggestion that shipped in Sonoma 14) is a small on-device autocomplete that works mostly in Apple’s own apps like Notes, Mail and Pages. Apple Intelligence Writing Tools is select-then-transform — Proofread, Rewrite, tone, Summarize — not continuous inline prediction. Shadowtype combines continuous multi-sentence inline ghost text with on-device rewrite in any app.
Is Apple Intelligence fully on-device?
Partly. Some requests run on-device, but Writing Tools can route to Private Cloud Compute, which runs on Apple servers — so it isn’t guaranteed to stay on your Mac or to work offline. Shadowtype runs every completion 100% on-device with zero network, so it works offline and your text never leaves the machine.
Do we support the same Macs?
Both require Apple Silicon. Apple Intelligence requires macOS 15.1 or later; Shadowtype runs on macOS 14 or later, so it covers a release Apple Intelligence doesn’t. Neither supports Intel Macs.
Can my own scripts, editors, and agents use Shadowtype's local model?
Yes. Shadowtype exposes an OpenAI-compatible HTTP endpoint on 127.0.0.1 with no key required, so your terminal, editor plugins, and AI agents can call /v1/chat/completions against the same on-device model. A built-in MCP server plugs into Claude Code and other MCP clients, and the BYOM picker loads any GGUF you drop in. Apple Intelligence keeps its on-device models locked to Apple’s own apps and Writing Tools — there’s no localhost API or MCP server for your code.
